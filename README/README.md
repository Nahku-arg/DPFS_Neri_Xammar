# Marketplace 

Proyecto final del curso de programación web (script 2) de DigitalHouse, que el mismo consiste en la creación de distintas maquetas de páginas teniendo en cuenta el feedback de los consumidores.

## Las páginas creadas fueron
- Home
- Página del producto a comprar
- Carrito de compras donde se puede agregar o eliminar el artículo
- Inicio de sesión
- Registro de usuario

## Tecnologías utilizadas
- HTML
- CSS
